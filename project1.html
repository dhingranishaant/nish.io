Project Product Price Tracking

PYTHON | VISUAL STUDIO CODE | BEAUTIFULSOUP

What I did

Introduction

This script tracks an online product's price and lets the user know when the price of the product selected by the user is in his/her budget range by sending an email to them. This script takes in an Amazon product's link, budget range and user's email address only. This works only when the script is left open to run.

Development

This project is concluded as a simple script in python language, written in Visual Studio Code. I learned web scraping through this project and used beautifulsoup to do so. This project is one of my favourite learnings as I completed this in my personal time.
