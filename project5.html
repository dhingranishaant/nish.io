Project Eazy Survey

ANGULAR | C# | SQL | VISUAL STUDIO CODE | VISUAL STUDIO | POSTMAN

What I did

Introduction

This project was made to learn Angular, SQL, and C# and how all these languages contribute to a well-working web app. This project has 2 different landing pages, one for an admin and the other one for a user. An admin can create surveys and post them, and user can take those surveys and submit them to be validated.

Development

This full-stack project is made using Angular, C#, and SQL. I used Visual Studio for backend, and Visual Studio Code for frontend.

Guidance

The only guidance I took for completing this short project of mine was through Youtube University. Youtube is a great place to learn coding and follow up with some tutorials to code with the instructor and learn. This project was a part of my internship at SOTI, where in the initial months a group project competetion was conducted.
