Project Flask Weather Dashboard

PYTHON, HTML, CSS | PYCHARM | FLASK

What I did

Introduction

This is one of my personal projects that I concluded in my reading break of my second undergraduate year. I built this web application using a weather API to collect weather data and showcase to the user. This web application lets the user to enter a favourable zip code inside United States of America to get weather details of that location.

Development

Made this project using python and beautified it using html and css. I utilized flask framework to complete this project and OpenWeatherMap API which allowed me to gather weather information of 50+ states of The United States using their state zip code.
