Project E-commerce Website

Java | Java Servlets | JSP | Apache Tomcat | SQL | Eclipse

What I did

Introduction

This project was a part of an academic course. It was to learn the basics of how an e-commerce website functions. This is a full-stack project lets users add their favourite tech gadgets to the cart and checkout (not actually).

Development

This project's frontend was built using JSP, using HTML and CSS with Bootstrap library. For algorithms, we used Java Servlets in Java. It was a group project, and our academic course required us to use Eclipse IDE for our project. For backend, we used SQL (MySQL).

Guidance

Guidance for this project was taken mainly from the course materials and a little from Youtube. We followed some tutorials on Youtube to learn more about Java Servlets and implement it efficiently in our project. Labs which were a part of the course were also a major help in order to conclude this project. Find more about the course here :- EECS 4413
